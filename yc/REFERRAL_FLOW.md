## Referral Flow Design

### Step 1: Activation Trigger
**When:** User activates (deploys first agent, runs 10+ executions)

**Action:** Show referral prompt

**Message:**
- **Headline:** "ðŸŽ‰ You're activated! Invite friends and get rewards"
- **Body:** "Share Agent Factory with friends. You both get free credits when they sign up."
- **CTA:** "Invite Friends" / "Get Referral Link"

**Timing:** Immediately after activation

**Frequency:** Once after activation

---

### Step 2: Referral Code Generation
**Action:** User clicks "Invite Friends"

**Process:**
- Generate unique referral code for user
- Create referral link (agentfactory.io/signup?ref=CODE)
- Show referral dashboard (track referrals, rewards)

**Features:**
- Unique referral code per user
- Shareable link (email, social, direct)
- Referral tracking dashboard
- Reward tracking

---

### Step 3: Referral Sharing
**Action:** User shares referral link

**Sharing Options:**
- **Email:** Pre-filled email template
- **Social:** Share on Twitter, LinkedIn
- **Direct:** Copy link, share directly
- **In-Product:** Share from dashboard

**Message Templates:**
- **Email:** "I've been using Agent Factory to deploy AI agents. You should try it! [Referral Link]"
- **Social:** "Just deployed my AI agent to production in minutes with Agent Factory. Try it: [Referral Link]"
- **Direct:** "Check out Agent Factory: [Referral Link]"

---

### Step 4: Friend Signs Up
**Action:** Friend clicks referral link and signs up

**Process:**
- Friend visits referral link
- Sees referral attribution ("You were referred by [Name]")
- Signs up with referral code applied
- Both user and friend get rewards

**Rewards:**
- **User (Referrer):** Free credits (e.g., 1 month Pro tier free)
- **Friend (Referee):** Free credits (e.g., 1 month Starter tier free)
- **Both:** Bonus credits when friend activates

---

### Step 5: Friend Activates
**Action:** Friend activates (deploys first agent, runs 10+ executions)

**Process:**
- Friend activates
- Both user and friend get additional rewards
- Track referral conversion
- Update referral dashboard

**Rewards:**
- **User (Referrer):** Additional credits (e.g., 1 more month Pro tier free)
- **Friend (Referee):** Additional credits (e.g., 1 more month Starter tier free)

---

## Referral Rewards

### Reward Structure

**Tier 1: Referral Signup**
- **Referrer:** 1 month Pro tier free (or equivalent credits)
- **Referee:** 1 month Starter tier free (or equivalent credits)

**Tier 2: Referral Activation**
- **Referrer:** 1 more month Pro tier free (or equivalent credits)
- **Referee:** 1 more month Starter tier free (or equivalent credits)

**Tier 3: Referral Conversion (Paid)**
- **Referrer:** 3 months Pro tier free (or equivalent credits)
- **Referee:** Already on paid tier

## Referral Tracking

### Metrics to Track
- **Referrals Sent:** Number of referral links shared
- **Referrals Clicked:** Number of referral links clicked
- **Referrals Signed Up:** Number of referrals who signed up
- **Referrals Activated:** Number of referrals who activated
- **Referrals Converted:** Number of referrals who converted to paid
- **Viral Coefficient:** Average referrals per user

### Referral Dashboard
- **User View:** Track referrals, rewards, status
- **Admin View:** Track all referrals, conversion rates, viral coefficient

---

## Referral Optimization

### High-Impact Optimizations
1. **Right Timing:** Show referral prompt after activation (not before)
2. **Clear Value:** Make rewards clear and compelling
3. **Easy Sharing:** Reduce friction in sharing process
4. **Track Everything:** Track all referral metrics

### Medium-Impact Optimizations
5. **Social Integration:** Easy social sharing
6. **Email Templates:** Pre-filled email templates
7. **Gamification:** Leaderboards, badges
8. **Personalization:** Personalize referral messages
