## Overview

This document maps the growth loops that drive Agent Factory's organic growth. Growth loops are self-reinforcing cycles where product usage drives more usage, creating sustainable, scalable growth.

---

## Primary Growth Loops

### Loop 1: Marketplace Flywheel

**Description:** Creators build blueprints ‚Üí Users discover and install blueprints ‚Üí Users become creators ‚Üí More blueprints ‚Üí More users

**Flow:**
1. **Creator builds blueprint** (e.g., teaching assistant)
2. **Blueprint published** to marketplace
3. **Users discover blueprint** (search, browse, recommendations)
4. **Users install blueprint** (free or paid)
5. **Users customize blueprint** for their needs
6. **Users become creators** (publish their version)
7. **More blueprints available** ‚Üí More users discover platform
8. **Loop repeats**

**Activation Points:**
- **Creator Activation:** First blueprint published
- **User Activation:** First blueprint installed
- **Conversion:** User ‚Üí Creator (10% target)

**Metrics:**
- **Blueprint Install Rate:** % of users installing blueprints (target: >30%)
- **Creator Conversion Rate:** % of users becoming creators (target: >10%)
- **Viral Coefficient:** Each creator brings X new users (target: >1.0)
- **Blueprint Discovery:** How users find blueprints (search, browse, recommendations)

**Optimization:**
- **Discovery:** Improve search, recommendations, featured blueprints
- **Installation:** Reduce friction (one-click install)
- **Creation:** Simplify blueprint creation process
- **Monetization:** Revenue share incentivizes creators

**Current Status:** ‚ö†Ô∏è Loop not yet active (marketplace not launched)

4. **Users create content** (blueprints, examples, tutorials)
5. **More content** ‚Üí More SEO opportunities

7. **Loop repeats**

**Activation Points:**

**Metrics:**

**Optimization:**
- **Content Quality:** High-value, SEO-optimized content
- **Distribution:** Share on Hacker News, Reddit, Twitter

### Loop 3: Referral ‚Üí User ‚Üí Referral

**Description:** User refers friend ‚Üí Friend signs up ‚Üí Friend refers others ‚Üí More users ‚Üí More referrals

**Flow:**
1. **User activates** (deploys first agent)
2. **Referral prompt** shown ("Invite friends, get credits")
3. **User shares referral code** (email, social, direct)
4. **Friend signs up** using referral code
5. **Both get rewards** (credits, features)
6. **Friend activates** ‚Üí Friend refers others
7. **Loop repeats**

**Activation Points:**
- **Activation:** User must activate before referral prompt
- **Reward:** Attractive reward (free credits, features)
- **Sharing:** Easy sharing (one-click, social)

**Metrics:**
- **Referral Rate:** % of users referring others (target: >20%)
- **Viral Coefficient:** Each user brings X new users (target: >1.0)
- **Referral Conversion:** % of referrals converting (target: >30%)
- **Time to Referral:** Days from activation to first referral

**Optimization:**
- **Timing:** Show referral prompt after activation
- **Reward:** Attractive reward (free month, credits)
- **Ease:** One-click sharing, pre-filled messages
- **Tracking:** UTM parameters, referral attribution

**Current Status:** ‚ö†Ô∏è Loop not yet active (referral system not implemented)
