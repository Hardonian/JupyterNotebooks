## On-Page SEO

### Title Tags

**Format:** Primary Keyword | Brand Name

**Examples:**
- "AI Agent Platform | Agent Factory"
- "Build AI Agents in Days | Agent Factory"
- "Education AI Platform | Agent Factory"

**Best Practices:**
- 50-60 characters
- Include primary keyword
- Include brand name
- Unique per page

---

### Meta Descriptions

**Format:** Value proposition + CTA

**Examples:**
- "Build production-ready AI agents in days, not months. FERPA compliant, education-focused platform. Start free trial."
- "AI agent platform for developers and educators. Pre-built blueprints, marketplace, production infrastructure. Try free."

**Best Practices:**
- 150-160 characters
- Include primary keyword
- Include value proposition
- Include CTA
- Unique per page

---

### Header Tags

**H1:** Primary keyword (one per page)  
**H2:** Secondary keywords (2-4 per page)  
**H3:** Supporting keywords (as needed)

**Example:**
```
H1: AI Agent Platform for Developers and Educators
H2: Build Production-Ready AI Agents
H2: Education-Focused Features
H2: Get Started Today
```

---

### Content Optimization

**Keyword Density:** 1-2%  
**Content Length:** 1,500+ words (for main pages)  
**Internal Linking:** 3-5 links per page  
**External Linking:** 2-3 quality links per page

**Best Practices:**
- Natural keyword usage
- Semantic keywords
- Related keywords
- LSI keywords

---

### URL Structure

**Format:** /category/keyword/

**Examples:**
- /platform/ai-agent-platform/
- /education/ferpa-compliant-ai/
- /developers/build-ai-agents/

**Best Practices:**
- Short URLs
- Include keywords
- Use hyphens
- Lowercase
- No special characters

---

### Image Optimization

**Alt Text:** Descriptive, include keywords  
**File Names:** Descriptive, include keywords  
**File Size:** Optimized (<200KB)  
**Format:** WebP preferred

**Examples:**
- Alt: "AI agent platform dashboard"
- File: "ai-agent-platform-dashboard.webp"
