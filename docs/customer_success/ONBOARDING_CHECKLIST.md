# Customer Onboarding Checklist

**Last Updated:** 2024-01-XX  
**Purpose:** Standardized onboarding checklist for Agent Factory Platform customers

---

## Pre-Onboarding

### Before Customer Signs

- [ ] Contract signed
- [ ] Payment processed
- [ ] Account created
- [ ] Welcome email sent
- [ ] Onboarding call scheduled

---

## Week 1: Initial Setup

### Day 1: Account Setup

- [ ] Welcome email sent
- [ ] Account access granted
- [ ] Login credentials shared
- [ ] Initial resources sent
- [ ] Welcome call scheduled

**Resources Sent:**
- Getting Started Guide
- Platform Overview
- Support Information
- Training Schedule

---

### Day 2-3: Welcome Call

- [ ] Welcome call completed
- [ ] Goals discussed
- [ ] Use cases identified
- [ ] Timeline established
- [ ] Success criteria defined

**Call Agenda:**
- Introduction (5 min)
- Goals discussion (10 min)
- Use case exploration (15 min)
- Platform overview (10 min)
- Q&A (10 min)
- Next steps (5 min)

---

### Day 4-5: Initial Training

- [ ] Platform training completed
- [ ] First agent created
- [ ] Basic features demonstrated
- [ ] Questions answered
- [ ] Next steps confirmed

**Training Topics:**
- Platform navigation
- Agent creation
- Basic configuration
- Testing and deployment
- Support resources

## Week 2: First Value

### Day 8-10: First Agent Deployment

- [ ] First agent configured
- [ ] Testing completed

**Checklist:**
- Agent created

### Day 11-14: First Check-In

- [ ] Progress review completed

## Success Criteria

### Onboarding Success

**Must Achieve:**
- First agent deployed within 14 days
- Positive feedback
- Health score 70+
- No critical issues

**Nice to Have:**
- First agent deployed within 7 days
- Excellent feedback
- Health score 80+
- Multiple agents deployed
