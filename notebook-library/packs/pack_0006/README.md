## Usage

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure environment (if needed):
   ```bash
   cp .env.example .env
   # Edit .env with your settings
   ```

3. Run the notebook:
   ```bash
   jupyter notebook main.ipynb
   ```
