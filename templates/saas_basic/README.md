# Basic SaaS Template

This template provides a minimal SaaS setup with:
- User authentication
- Basic multi-tenancy
- Simple billing integration
- API key management

## Setup

1. Copy this template to your project
2. Configure environment variables
3. Run migrations
4. Start the server

## Structure

```
saas_basic/
├── auth.py          # Authentication logic
├── models.py        # Database models
├── routes.py        # API routes
├── billing.py       # Billing integration
└── config.py        # Configuration
```
